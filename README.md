# Modern E-Commerce Store

A production-ready frontend e-commerce application built with React, Tailwind CSS, and Context API.

## Features

- **Storefront**: Responsive Home, Shop, and Product Details pages.
- **Cart System**: Full cart management with persistence.
- **Wishlist**: Save items for later.
- **Authentication**: Simulated Login/Register flows.
- **Search & Filtering**: Filter by category, price, and search products.
- **Production Ready**: Clean code architecture, toasts, reusable components.

## Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **React Router DOM**
- **Context API** (State Management)
- **Lucide React** (Icons)
- **LocalStorage** (Persistence)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Folder Structure

- `src/components`: UI primitives and feature components.
- `src/context`: Global state providers (Auth, Cart, Wishlist, Toast).
- `src/pages`: Route components.
- `src/data`: Mock data.
